Vpsproxy

近期在使用国内vps时遇到了访问github，或者steamcmd等一些外网资源困难的问题，在网上搜索到了一些解决办法，故此记录 首先是proxychain 安装proxychain sudo apt update sudo apt install proxychains 配置sock5代理 sudo nano /etc/proxychains.conf 找到最后一行的示例（通常是 socks4 127.0.0.1 9050），改成： socks5 ip port 最后只需要在正常使用命令之前加上proxychain就可以正常访问了。 搭配frp可以很方便将本地电脑的socks代理共享给vps
